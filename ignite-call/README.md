# Project

## Description

Just a simple scheduler app

## Installation

### Versions

```
npm: 9.4.2
Docker version 20.10.17, build 100c701
```

### Local

```bash
npm install
```

### Docker

#### Running

```bash
docker run --name mysql -e MYSQL_ROOT_PASSWORD=docker -p 3306:3306 mysql:latest
```

#### Start container

```bash
docker start mysql
```

#### Stop container

```bash
docker stop mysql
```
